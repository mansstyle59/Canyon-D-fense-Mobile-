## Installation et exécution

1. **Installer les dépendances :**  
   ```bash
   npm install
   ```

2. **Configuration de la clé API :**  
   Ajoutez votre clé Gemini API dans le fichier `.env.local` :  
   ```env
   GEMINI_API_KEY=VOTRE_CLE_API
   ```

3. **Démarrer l'application :**  
   ```bash
   npm run dev
   ```

Votre application sera accessible à l'adresse suivante : `http://localhost:3000`
